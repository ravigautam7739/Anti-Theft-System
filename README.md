# 🛡️ Anti-Theft System (Mouse Movement Detection)

A Python-based **anti-theft security system** that monitors mouse movement and automatically locks the computer if any movement is detected.

This project simulates a **basic workstation security alarm system** using automation and system-level controls.

---

# 🚀 Features

✔ Detects mouse movement in real-time
✔ Automatically locks the system on movement
✔ Lightweight background monitoring
✔ Simple and fast security automation
✔ Useful for workstation protection

# 🧠 How It Works

1. Program stores current mouse position
2. Continuously monitors cursor movement

### 📌 Logic:

* No movement → System stays armed
* Mouse movement detected →

  ```id="anti4"
  Movement detected!
  Locking system...
  ```

3. Windows lock screen is triggered automatically
